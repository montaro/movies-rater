movies-rater
============

Scans a directory of movies sub-directories and fetch the ratings from a web service and sort them by rating

Usage Notes
===========

```
$./movies-rater.py /home/arefaey/movies
```

Dependencies
============

* os
* httplib2
* json
* operator
* urllib
* sys
* re
